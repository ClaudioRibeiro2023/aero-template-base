# Internacionalização (i18n)

## Stack

- **i18next** + **react-i18next** + **i18next-browser-languagedetector**
- 2 idiomas: `pt-BR` (padrão) e `en-US`
- 4 namespaces: `common`, `dashboard`, `tasks`, `settings`
- 120+ chaves de tradução

## Uso

```tsx
import { useTranslation } from 'react-i18next'

function MyComponent() {
  const { t } = useTranslation('common')
  return <h1>{t('welcome')}</h1>
}
```

## LanguageSwitcher

```tsx
import { LanguageSwitcher } from '@/components/common/LanguageSwitcher'
;<LanguageSwitcher /> // Detecta idioma do browser + persiste em localStorage
```

## Arquivos de Tradução

```
apps/web/src/locales/
├── pt-BR/
│   ├── common.json
│   ├── dashboard.json
│   ├── tasks.json
│   └── settings.json
└── en-US/
    ├── common.json
    ├── dashboard.json
    ├── tasks.json
    └── settings.json
```

## Adicionar Nova Chave

1. Adicione a chave em `pt-BR/<namespace>.json`
2. Adicione a tradução em `en-US/<namespace>.json`
3. Use `t('namespace:key')` ou `useTranslation('namespace')` + `t('key')`
